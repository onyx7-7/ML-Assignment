# Student Mental Health Survey - Dataset Description

## Dataset Source
- **Platform:** Kaggle
- **Link:** https://www.kaggle.com/datasets/shariful07/student-mental-health
- **Author:** Shariful Islam

---

## Number of Rows and Columns
- **Rows:** 101 student responses
- **Columns:** 11 columns

---

## Description of Features (Columns)

| Column | Description |
|--------|-------------|
| `Timestamp` | Date and time of survey response |
| `Choose your gender` | Gender of the student (Male / Female) |
| `Age` | Age of the student |
| `What is your course?` | The student's field of study / major |
| `Your current year of Study` | Academic year (Year 1 – Year 4) |
| `What is your CGPA?` | Current cumulative GPA range |
| `Marital status` | Whether the student is married (Yes / No) |
| `Do you have Depression?` | Self-reported depression status (Yes / No) |
| `Do you have Anxiety?` | Self-reported anxiety status (Yes / No) |
| `Do you have Panic attack?` | Self-reported panic attack status (Yes / No) |
| `Did you seek any specialist for a treatment?` | Whether the student sought professional help (Yes / No) |

---

## Purpose of Using This Dataset

The purpose of this dataset is to explore the mental health status of university students and understand the patterns and factors associated with depression, anxiety, and panic attacks among students.

Through Exploratory Data Analysis (EDA), this project aims to:
- Identify which student groups are most affected by mental health issues
- Understand the relationship between academic performance (CGPA), year of study, and mental health
- Explore gender differences in mental health outcomes
- Investigate how many students seek professional treatment
- Provide insights that could help universities better support student wellbeing
